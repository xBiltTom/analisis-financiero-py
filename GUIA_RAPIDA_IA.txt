┌──────────────────────────────────────────────────────────────────────┐
│ PASO 1: Iniciar la Aplicación                                       │
└──────────────────────────────────────────────────────────────────────┘

   En la terminal, ejecuta:
   
   streamlit run analizador_financiero.py


┌──────────────────────────────────────────────────────────────────────┐
│ PASO 2: Cargar Archivos Financieros                                 │
└──────────────────────────────────────────────────────────────────────┘

   1. Usa el botón "Browse files" en la barra lateral
   2. Selecciona archivos XLS con estados financieros
   3. Solo archivos del año 2010 en adelante son compatibles
   4. Puedes cargar múltiples archivos de diferentes años


┌──────────────────────────────────────────────────────────────────────┐
│ PASO 3: Navegar a la Pestaña de Ratios                              │
└──────────────────────────────────────────────────────────────────────┘

   1. Busca la pestaña "Vista Consolidada - Ratios"
   2. Haz clic para abrir
   3. Verás los ratios calculados en tablas y gráficos


┌──────────────────────────────────────────────────────────────────────┐
│ PASO 4: Generar Análisis con IA                                     │
└──────────────────────────────────────────────────────────────────────┘

   1. Desplázate hacia abajo hasta ver:
      
      🤖 Análisis Inteligente con IA
      ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   
   2. Haz clic en el botón:
      
      [🔍 Generar Análisis con IA]
   
   3. Espera unos segundos mientras la IA analiza los datos


┌──────────────────────────────────────────────────────────────────────┐
│ PASO 5: Revisar el Análisis                                         │
└──────────────────────────────────────────────────────────────────────┘

   El análisis incluirá:
   
   ✅ Análisis de Liquidez
      → Evalúa capacidad de pago a corto plazo
   
   ✅ Análisis de Endeudamiento
      → Evalúa nivel de riesgo financiero
   
   ✅ Análisis de Rentabilidad
      → Evalúa capacidad de generar ganancias
   
   ✅ Análisis de Actividad
      → Evalúa eficiencia operativa
   
   ✅ Tendencias Observadas
      → Identifica patrones entre años
   
   ✅ Conclusiones y Recomendaciones
      → Sugerencias estratégicas


┌──────────────────────────────────────────────────────────────────────┐
│ PASO 6: Descargar el Análisis (Opcional)                            │
└──────────────────────────────────────────────────────────────────────┘

   1. Encuentra el botón:
      
      [📥 Descargar Análisis de IA (TXT)]
   
   2. Haz clic para descargar
   3. El archivo se guardará con el nombre:
      analisis_ia_ratios_[NOMBRE_EMPRESA].txt


╔══════════════════════════════════════════════════════════════════════╗
║                          RATIOS ANALIZADOS                           ║
╚══════════════════════════════════════════════════════════════════════╝

 📊 LIQUIDEZ (2 ratios)
 ├─ Liquidez Corriente
 └─ Prueba Ácida

 💰 ENDEUDAMIENTO (2 ratios)
 ├─ Razón de Deuda Total
 └─ Razón Deuda/Patrimonio

 📈 RENTABILIDAD (3 ratios)
 ├─ Margen Neto
 ├─ ROA (Return on Assets)
 └─ ROE (Return on Equity)

 ⚡ ACTIVIDAD (3 ratios)
 ├─ Rotación de Activos Totales
 ├─ Rotación de Cuentas por Cobrar
 └─ Rotación de Inventarios

mantiene una posición de liquidez favorable...

║                    CONSEJOS Y RECOMENDACIONES                        ║
╚══════════════════════════════════════════════════════════════════════╝

 💡 CONSEJO 1: Carga múltiples años
    → Más años = Mejor análisis de tendencias
    → Recomendado: Mínimo 3 años consecutivos

 💡 CONSEJO 2: Verifica los datos antes
    → Revisa las tablas de ratios primero
    → Asegúrate de que los valores sean correctos

 💡 CONSEJO 3: Usa el análisis como guía
    → El análisis de IA es una herramienta de apoyo
    → Complementa con tu conocimiento del negocio

 💡 CONSEJO 4: Descarga para referencia
    → Guarda el análisis para reportes
    → Comparte con el equipo financiero


╔══════════════════════════════════════════════════════════════════════╗
║                    SOLUCIÓN DE PROBLEMAS                             ║
╚══════════════════════════════════════════════════════════════════════╝

 ❓ ¿El botón no aparece?
    → Verifica que hayas cargado archivos POST-2010
    → Asegúrate de estar en la pestaña correcta

 ❓ ¿El análisis tarda mucho?
    → Normal: puede tomar 5-10 segundos
    → Si tarda más de 30 segundos, recarga la página

 ❓ ¿Aparece un error?
    → Verifica tu conexión a internet
    → Intenta generar el análisis nuevamente

 ❓ ¿El análisis no tiene sentido?
    → Verifica que los datos de entrada sean correctos
    → Reporta el problema con los detalles
